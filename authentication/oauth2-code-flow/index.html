OAuth2 Code Flow Sample

This is a demo on how to get a token using the authorization code flow.

This page redirects you to the login website, to authenticate and retrieve a code.

Steps:
  1. Copy the sources to a web server running on http://localhost/openapi-samples-js/authentication/oauth2-code-flow/
  2. Make sure PHP is installed on that server
  3. Enter app id and secret in the PHP sources
  4. Open http://localhost/openapi-samples-js/authentication/oauth2-code-flow/index.html
