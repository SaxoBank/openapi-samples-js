Steps:
  1. Copy the sources to a web server running on http://localhost/openapi-samples-js/authentication/oauth2-pkce-flow/

  3. Enter app id and secret in the PHP sources
  4. Open http://localhost/openapi-samples-js/authentication/oauth2-pkce-flow/index.html
