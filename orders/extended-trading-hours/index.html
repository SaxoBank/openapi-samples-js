Extended Trading Hours (ETH)

Place orders outside regular AutomatedTrading session

← Back to main page View on GitHub

Execute orders in the Pre Market and After Hours trading sessions

This example shows how place an order for the extended trading hours. This is done by adding the ExecuteAtTradingSession enum set to "All". If the exchange supports this and AutomatedTrading session is not running, the order is moved to either pre-, or post trading session.
The default value is "Regular".
ETH trading is not available for all accounts. It must be enabled for an account. More info on help.saxo.
