Block orders and trade allocation

This example shows how place a block order and allocate the position into a number of sub-accounts. Block trading is only available for specially configured accounts. To get an account configured for block trading, the account owner must contact their Saxo Bank account manager.
More info on block orders: https://www.developer.saxo/openapi/learn/allocation-keys
